Calculating Mean and Variance Online

In the previous assignment, we explored how to calculate the mean and variance from a complete dataset. However, in real-world applications— especially in fields like cybersecurity, data streaming, and financial analysis— data often arrives incrementally. It is therefore essential to compute statistical measures online, updating them efficiently each time a new value is observed.

1. What Does “Online” Mean?

An online algorithm computes statistics dynamically as new data points arrive, without storing the entire dataset or recomputing everything from scratch. This approach is particularly useful for large datasets or continuous data streams.

2. Online Mean Computation

The online mean can be updated using a simple recurrence relation: 


meanₙ = meanₙ₋₁ + (xₙ − meanₙ₋₁) / n

This formula updates the mean each time a new data point xₙ is added, without needing to reprocess all previous values. It’s a constant-time update, which makes it efficient and ideal for real-time analysis. 


// Online mean in JavaScript
let mean = 0;
let count = 0;

function updateMean(newValue) {
  count++;
  mean += (newValue - mean) / count;
  return mean;
}

3. Online Variance Computation

The variance can also be computed online using Welford’s algorithm, which is both numerically stable and efficient: 


let count = 0;
let mean = 0;
let M2 = 0; // sum of squared deviations

function updateVariance(newValue) {
  count++;
  const delta = newValue - mean;
  mean += delta / count;
  M2 += delta * (newValue - mean);
  const variance = M2 / count;
  return { mean, variance };
}

Each new observation updates the mean and the accumulated squared deviations M2. From this, the variance can be computed efficiently without revisiting old data points.

4. Why Online Computation Matters

Online methods are fundamental in systems that handle continuous data or large-scale analytics, such as:

  • Real-time anomaly detection in cybersecurity
  • Streaming financial indicators
  • Sensor networks and IoT analytics
  • Machine learning model updates without retraining

These applications rely on the ability to update metrics on-the-fly without the computational overhead of recomputing from scratch.
