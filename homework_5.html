Measures of Location and Dispersion

This assignment explores two fundamental concepts in statistics: measures of location (or central tendency) and measures of dispersion. Together, these measures describe how data is distributed — both where it is centered and how spread out it is.

1. Measures of Location (Central Tendency)

Measures of location indicate the center or typical value around which data points cluster. They summarize a dataset with a single representative number, giving a sense of its “average” behavior. The most common measures include:

  • Arithmetic Mean: The most widely used measure of central tendency, calculated by summing all values and dividing by their count. 
    
// JavaScript example:
const values = [2, 4, 6, 8, 10];
const mean = values.reduce((a, b) => a + b, 0) / values.length;
console.log(mean); // 6
    Useful when data is symmetrically distributed without extreme outliers.
  • Median: The middle value when data is sorted. It’s less sensitive to outliers and skewed distributions. 
    
const values = [1, 3, 100];
values.sort((a, b) => a - b);
const median = values.length % 2
  ? values[(values.length - 1) / 2]
  : (values[values.length / 2 - 1] + values[values.length / 2]) / 2;
console.log(median); // 3
    Preferred for skewed distributions (e.g., income data).
  • Mode: The most frequently occurring value in a dataset. 
    
const values = [1, 2, 2, 3, 3, 3, 4];
const freq = {};
values.forEach(v => freq[v] = (freq[v] || 0) + 1);
const mode = Object.keys(freq).reduce((a, b) => freq[a] > freq[b] ? a : b);
console.log(mode); // 3
    Useful for categorical or discrete data (e.g., most common product sold).

Each of these measures tells a slightly different story about the same dataset. For example, the mean captures the overall average, while the median provides a better picture when extreme values distort the average. The mode reflects the most typical value, often used in qualitative analyses.

2. Measures of Dispersion (Variability)

While measures of location describe where the data tends to cluster, measures of dispersion indicate how much the data spreads out around that center. Without dispersion, two datasets could share the same mean but behave very differently.

  • Range: The simplest measure, defined as the difference between the maximum and minimum values. 
    
const values = [4, 7, 9, 12, 15];
const range = Math.max(...values) - Math.min(...values);
console.log(range); // 11
    Quick but highly sensitive to outliers.
  • Variance: The average of squared deviations from the mean. It quantifies how far individual points lie from the center on average. 
    
const values = [4, 7, 9, 12, 15];
const mean = values.reduce((a, b) => a + b, 0) / values.length;
const variance = values.reduce((acc, v) => acc + (v - mean) ** 2, 0) / values.length;
console.log(variance);
    Used extensively in probability, finance, and quality control.
  • Standard Deviation: The square root of the variance. It expresses dispersion in the same units as the original data. 
    
const stdDev = Math.sqrt(variance);
console.log(stdDev);
    Preferred when comparing variability across datasets with similar units.
  • Interquartile Range (IQR): The difference between the 75th percentile (Q3) and the 25th percentile (Q1). It represents the range of the middle 50% of data. 
    
// Simple percentile approximation:
function percentile(arr, p) {
  arr.sort((a, b) => a - b);
  const i = (arr.length - 1) * p;
  const lower = Math.floor(i), upper = Math.ceil(i);
  return arr[lower] + (arr[upper] - arr[lower]) * (i - lower);
}
const values = [1, 3, 5, 7, 9, 11];
const IQR = percentile(values, 0.75) - percentile(values, 0.25);
console.log(IQR);
    Useful when detecting outliers or for non-normal distributions.

3. Why We Need Both Location and Dispersion

Measures of location tell us the central value, while measures of dispersion show how tightly or loosely the data is spread around it. One without the other can lead to misleading interpretations.

For instance, consider two exam groups with the same mean score of 70: one has scores between 68 and 72 (low dispersion), while another has scores between 30 and 110 (high dispersion). The mean alone hides this crucial difference.

In summary, measures of location (mean, median, mode) describe where data points cluster, and measures of dispersion (variance, standard deviation, range, IQR) describe how spread out they are. Both are essential for accurate and meaningful statistical analysis.
