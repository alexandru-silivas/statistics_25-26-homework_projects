Homework 7

Security updates, random walks and binomial convergence (n = 50 weeks)

Model: each week there are m independent attackers, each with breach probability p. The server is considered breached in a week if at least one attacker succeeds. Assign +1 if the server remains secure that week, −1 if breached. Over n weeks the cumulative score is the sum of those ±1 steps.

Weekly secure probability: s = (1 − p)m. Number of secure weeks K ∼ Binomial(n, s). Final score S = 2K − n; possible scores are {−n, −n+2, …, n−2, n}. The theoretical probability:

P(S = 2k − n) = C(n,k) · sk · (1−s)n−k
